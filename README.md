# Resume Scanner

A Python project to automatically scan and compare resumes against job descriptions, extract relevant skills, and generate a match score.

## Features
- Extracts keywords from job descriptions
- Compares resumes against these keywords
- Generates matched skills, missing skills, and a percentage match score
- Supports PDF resumes

## Skills Used
Python, PyPDF2, Text Processing, File Handling, Sequence Matching

## How to Run
1. Place all resumes (PDFs) in the `Resumes` folder.
2. Run `Resume_Scanner.py` and paste the job description when prompted.
3. See matched skills and match score for each resume.
